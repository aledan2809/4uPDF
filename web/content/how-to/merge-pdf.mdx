## Advanced Merging Options

**Page Range Selection**: Extract specific pages from each PDF before merging. For example, include only pages 1-5 from document A and pages 10-15 from document B.

**Bookmark Preservation**: Maintain existing bookmarks from source PDFs to create a navigable table of contents in your merged document.

**Compression Control**: Choose compression levels to balance file size and quality. Use no compression for maximum quality or high compression for smaller file sizes.

## Tips for Better PDF Merging

**Name Files Logically**: Use numbered prefixes (01-file, 02-file) to ensure files appear in the correct order when uploading multiple documents.

**Check File Quality**: Before merging, open each PDF to verify it contains the expected content and isn't corrupted or password-protected.

**Optimize File Sizes**: If your merged PDF is too large, compress individual files before merging using our [PDF compression tool](/how-to-compress-pdf).

**Preview Before Distributing**: After merging, scroll through the entire document to verify correct page order and content.
