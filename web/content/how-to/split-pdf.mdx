# How to Split PDF Files Online

Divide large PDF files into smaller, more manageable documents with 4uPDF's free split tool. Extract specific pages, split by page count, or use intelligent OCR-based splitting for scanned documents.

## Why Split PDF Files?

**File Size Management**: Break large PDFs into smaller files that are easier to email, upload, and share without exceeding size limits.

**Document Organization**: Separate multi-document scans into individual files for better organization and searchability.

**Selective Sharing**: Extract only the pages you need to share, keeping confidential sections private.

**Performance**: Smaller files open faster and are easier to work with, especially on mobile devices or slower connections.

## How to Split PDFs: Multiple Methods

### Method 1: Extract Specific Pages

Select exact pages or page ranges to extract as separate files. This is perfect when you know which pages you need.

**Example**: From a 100-page PDF, extract pages 1-5, 10-15, and 20-25 as three separate files.

**Steps**:
1. Upload your PDF (up to 100MB on free tier)
2. Choose "Extract Specific Pages" mode
3. Enter page ranges (e.g., "1-5, 10-15, 20-25")
4. Download extracted PDFs as a ZIP file

### Method 2: Split by Page Count

Divide your PDF into chunks of equal page counts. Great for breaking large documents into manageable sections.

**Example**: Split a 300-page PDF into 30 files of 10 pages each.

**Steps**:
1. Upload your PDF
2. Select "Split by Page Count"
3. Enter pages per file (e.g., 10)
4. Download all split files as a ZIP archive

### Method 3: Split at Blank Pages

Automatically detect blank separator pages and split there. Perfect for batch scans with separators.

**Example**: You scanned 50 invoices with blank pages between each one. The tool automatically creates 50 separate invoice files.

**Steps**:
1. Upload your scanned PDF
2. Choose "Split at Blank Pages"
3. Enable "Remove blank pages" if you don't want them in results
4. Download separated documents

### Method 4: OCR-Based Intelligent Splitting

Use OCR to detect document boundaries based on text patterns. This is the most advanced method for recurring document types like invoices, contracts, or reports.

**Example**: Split a 200-page scan containing 50 invoices by detecting "Invoice Number:" at the start of each invoice.

**Steps**:
1. Upload your PDF
2. Enable "OCR-Based Splitting"
3. Define text pattern (e.g., "Invoice Number:")
4. Configure automatic naming using extracted values
5. Download intelligently split and named files

**Preview Before Splitting**: Use the thumbnail preview to verify page boundaries before processing.

**Use OCR for Scanned Documents**: Enable OCR during splitting to make resulting files searchable, not just images.

**Choose Appropriate Split Method**: Use manual ranges for one-time jobs, blank page detection for separator-based scans, and OCR splitting for recurring document types.

**Verify Results**: After splitting, open a few random files to ensure splitting occurred at the correct boundaries.

**Organize Immediately**: Create folder structures and apply naming conventions right after splitting to maintain organization.
