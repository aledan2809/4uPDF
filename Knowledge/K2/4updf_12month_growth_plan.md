# 4uPDF 12‑Month Traffic Growth Plan

Goal: reach \~100k monthly visitors with minimal paid advertising.

Strategy pillars: 1. Publish many SEO tool pages 2. Target long‑tail
keywords 3. Build topical authority around PDF processing 4. Launch
differentiating automation tools

Budget assumption: \~200€ / month (mostly hosting + small outreach)

------------------------------------------------------------------------

# Month 1 --- SEO Foundation

Tasks - Create dedicated pages for the core tools: /merge-pdf /split-pdf
/compress-pdf /pdf-to-word /word-to-pdf /jpg-to-pdf /pdf-to-jpg

Technical SEO - sitemap.xml - robots.txt - canonical tags - page
titles + meta descriptions

Content - Add FAQ sections for each tool.

Expected traffic \~200--500 visits/month.

------------------------------------------------------------------------

# Month 2 --- Conversion Tools
