# 4uPDF Pricing Strategy

Goal: maximize conversions while keeping the free tier attractive for
SEO traffic.

## Free Plan

Target: casual users and SEO traffic

Limits: - Max file size: 50 MB - Max pages per file: 200 - Daily
processing limit - Ads displayed - Basic tools only

Purpose: Drive traffic and Ad revenue.

------------------------------------------------------------------------

## Bronze Plan

Price suggestion: 3--4 € / month

Limits: - Max file size: 150 MB - Max pages: 500 - No ads - Faster
processing - Access to basic batch tools

Target users: Frequent individual users.

------------------------------------------------------------------------

## Silver Plan

Price suggestion: 7--9 € / month

Limits: - Max file size: 300 MB - Max pages: 1000 - Batch processing -
Priority queue - Access to smart tools (OCR split, rename)

Target users: SMEs and power users.

------------------------------------------------------------------------

## Gold Plan

Price suggestion: 15--19 € / month

Limits: - Max file size: 500 MB - Unlimited pages - Automation
workflows - Archive processor - API access

Target users: Companies processing large document archives.

------------------------------------------------------------------------

## Conversion Strategy

-   Free plan visible and generous
-   Show upgrade prompts only when limits are reached
-   Offer annual discount (20--30%)
