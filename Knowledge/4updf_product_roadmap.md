# 4uPDF Product Roadmap

## Vision

Build a hybrid platform combining high‑traffic PDF tools with
intelligent document automation features useful for SMEs.

------------------------------------------------------------------------

## Phase 1 -- Core SEO Tools (Traffic Engine)

These tools generate the majority of global search traffic.

-   Merge PDF
-   Split PDF
-   Compress PDF
-   PDF to Word
-   Word to PDF
-   JPG to PDF
-   PDF to JPG

Goal: acquire organic traffic through common searches.

------------------------------------------------------------------------

## Phase 2 -- Standard Editing Tools

Match the functionality users expect from major PDF platforms.

-   Rotate PDF
-   Delete pages
-   Extract pages
-   Sign PDF
-   Watermark PDF
-   Protect PDF
-   Unlock PDF

Goal: reach feature parity with major competitors.

------------------------------------------------------------------------

## Phase 3 -- Smart Tools (Differentiation)

Unique features based on OCR and document analysis.

-   Split by text pattern
-   Split by invoice/order number
-   Split by barcode
-   Auto rename PDFs from detected text
-   Document type detection (invoice / contract / delivery note)

Goal: position 4uPDF as a smart automation platform.

------------------------------------------------------------------------

## Phase 4 -- Automation Workflows

Advanced document processing pipelines.

Examples:

-   Archive processor (scan → detect → split → rename)
-   Bulk invoice extractor
-   Document classification for large scanned archives
-   Automatic document sorting

Goal: SME adoption and higher‑value subscriptions.
